# Loan Default Analysis Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyses loan default risk based on income, employment type, loan purpose and age group.

The project helps identify financial behaviour patterns and potential risk indicators useful for banks and financial institutions.

## Key Insights
- Loan amount by purpose
- Average income by employment type
- Default rate (%) by employment category
- Default trend by year
- Loan amount by age group

## Tools Used
- Power BI
- Data Analysis
- Data Visualization
- Financial Risk Analysis
